# Smart Energy Analytics

A state-of-the-art energy consumption prediction and dashboard built with React and LSTM Deep Learning.

## Features
- **AI Predictions**: Precise hourly energy consumption forecasting using LSTM models.
- **Peak Analysis**: Automated classification of usage into Peak and Off-Peak categories.
- **Dynamic Visualizations**: Real-time charts comparing average vs. predicted usage.
- **Actionable Insights**: Smart recommendations to optimize energy usage and reduce costs.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts, Lucide Icons.
- **Backend**: Python, Flask.
- **Machine Learning**: TensorFlow (LSTM), Scikit-learn.

## Getting Started
### Prerequisites
- Node.js & npm
- Python 3.8+

### Installation
1. Clone the repository:
   ```sh
   git clone <repository-url>
   ```
2. Setup and run the backend:
   ```sh
   cd backend
   pip install -r requirements.txt
   python app.py
   ```
3. Setup and run the frontend:
   ```sh
   cd energy-insights-dashboard
   npm install
   npm run dev
   ```
